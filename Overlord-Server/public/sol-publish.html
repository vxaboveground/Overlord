Solana Memo Publisher

Publish an encrypted server URL as a Solana memo transaction. Clients configured with Solana lookup will query this address to find the server.

Your private key is never stored — it's used in-memory only for signing and discarded immediately.

The server URL that clients will connect to. Multiple URLs can be separated by newlines.

How it works

  • The server URL is encrypted with AES-256-GCM using SHA-256 of your server's Agent Token as the key.
  • The encrypted memo is published as a Solana memo transaction from your wallet.
  • Clients built with Solana mode query your wallet address for the latest memo and decrypt it.
  • Publishing costs approximately 0.000005 SOL (~$0.001).
  • To update the server URL, just publish a new memo — clients always use the newest one.
