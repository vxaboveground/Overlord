How to Use

Start the Proxy

Click "Start Proxy" to launch a SOCKS5 proxy server on the agent machine. The proxy will listen on localhost (127.0.0.1) at the specified port.

Configure Your Applications

Configure your browser or applications on the agent machine to use the SOCKS5 proxy at 127.0.0.1:[port]. This will route traffic through the agent's network.

Use Your Agent's IP

All traffic routed through the proxy will appear to originate from the agent's IP address and network location.

Note

The proxy runs directly on the agent machine. You need to configure applications on that machine to use it, or set up port forwarding/tunneling to access it remotely.
